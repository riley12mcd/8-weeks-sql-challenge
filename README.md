# 8 Week SQL Challenge

## About the project
The 8 Week SQL Challenge is a series of realistic business case studies created by Danny Ma, 
designed to help data professionals build and demonstrate their SQL skills through real world 
problems. Each case study is built around a fictional business scenario with actual data to 
query and analyze. You can find the original challenge at https://8weeksqlchallenge.com.
